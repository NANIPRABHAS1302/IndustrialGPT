## 📌 Executive Package Overview

Welcome to the official submission package for **IndustrialGPT**. This repository contains the complete, production-grade technical documentation, architecture diagrams, user manuals, presentation slides, database schemas, API references, and build automation tools required for judging in the ET AI Hackathon.

IndustrialGPT fuses **multi-modal OCR document processing**, **ChromaDB vector similarity search**, a **Neo4j Knowledge Graph**, and a **telemetry-driven Remaining Useful Life (RUL) predictive algorithm** into a unified operational brain.
